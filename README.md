# 🚀 Watt, Solar System Calculator - Version 1.0

### ⚠️ Important Notes
*   **Initial Release:** This is the first release of the application. Currently, all services requiring authentication (Login, Company portals, etc.) are active for **testing purposes only** and are not yet ready for production use.
*   **Regional Availability:** Please note that company operations and PV parts shopping features are currently exclusive to **Iraq**.

---

### 🛠 Key Features

*   **Home Solar System Calculator:** 
    *   **Fast Mode:** Quick estimates based on average consumption.
    *   **Detailed Mode:** Precise calculations based on specific home appliances and usage patterns.
*   **PV Module Calculator:** Professional estimation of solar panel performance and output.
*   **Inverter Sizing:** Support for multiple configurations:
    *   Hybrid, Off-Grid, and On-Grid systems.
    *   Three-phase and Single-phase options.
*   **Wire Sizing:** Accurate gauge calculations for both **AC** and **DC** wiring to ensure safety and efficiency.
*   **Solar Pump Calculator:** Specialized tools for sizing solar-powered water pumping systems.
*   **Angle and Direction:** Optimization guide for the best tilt angle and orientation based on your location (In Progress - Not yet complete).
